Your To-Do App

A clean, modern, and personalized To-Do application that allows users to enter their name as the header and manage their daily tasks with a beautifully designed interface.
Built for simplicity, speed, and a more personal productivity experience.

Features

Personalized Header — User enters their name and it appears at the top

 Add Tasks Easily — Quick task input with smooth interaction

Modern UI — Clean, beautiful, and responsive design

 Mobile-Friendly — Works on all screen sizes

 Delete Tasks — Remove any task with one click

CSS
